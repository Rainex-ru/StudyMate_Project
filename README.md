# StudyMate Bot

Telegram бот для помощи школьникам в выборе учебного заведения с использованием ИИ.

## Функционал

- Интерактивный опрос для сбора данных о предпочтениях
- Анализ баллов ОГЭ/ЕГЭ
- Рекомендации вузов и специальностей через GigaChat AI
- Возможность дополнить ответы

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Настройте переменные окружения в файле `.env`:
```
BOT_TOKEN=ваш_токен_бота
GIGA_CREDENTIALS=ваш_ключ_авторизации_gigachat
```

Можно использовать также `GIGACHAT_CREDENTIALS` или `GIGA_AUTH_KEY`.

В `.env` должен передаваться именно Authorization key из GigaChat API Setup, не Client ID.

3. Запустите бота:
```bash
python main.py
```

## Структура проекта

- `main.py` - основной файл бота
- `config.py` - конфигурация
- `handlers/` - обработчики сообщений
- `keyboards/` - клавиатуры
- `services/` - сервисы ИИ

## Использование

1. Отправьте `/start` боту
2. Выберите "📝Пройти опрос"
3. Ответьте на вопросы
4. Получите рекомендации от ИИ
5. При необходимости дополните ответ
