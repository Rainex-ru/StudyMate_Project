Как пользоваться
  1. Войдите через Telegram или введите тот же числовой ID, что используете в боте.
  2. Инструменты повторяют логику бота: поиск по городу, опрос по баллам, короткий тест, уточнение ответа.
  3. История подтягивается из локальной базы SQLite — это те же записи, что сохраняет бот.
